## Technology Stack

- Language: TypeScripts
- Framework: Fastify
- Database: MySQL 
- ORM: Prisma
- Authentication: JWT
- API Communication: RESTful API
- Containerization: Docker & Docker Compose
- API Documentation: Postman , Swagger
